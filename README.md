# SimpleVirtualAssistant
A simple assistant that can run in the background and do tasks that you tell it by voice, such as searching on google, opening different apps,
telling jokes, and using wolfram alpha for math. 
project is based on tutorials i found on youtube 
will need to install a number of packages to import them for it to run such as speech_recognition, puaudio, gTTS, _tkinter, webbrowser, os, spotify_local, and others. 
written in python 3.7 (64-bit) in Visual Studio
